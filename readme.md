## DESKRIPSI SOAL
nput : Koefisien $a,b,c$ dari polinomial $P_2(x)=ax^2+bx+c$ dimana $a\neq0$.

Output :

Diskriminan
Akar-akar dari polinomial
Titik stasioner
Kecekungan Kurva
Nilai maksimum/minimum lokal
